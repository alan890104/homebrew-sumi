# Homebrew Tap for Sumi

[Sumi](https://github.com/alan890104/sumi) is a system-wide speech-to-text transcription app for macOS (Apple Silicon).

## Installation

```bash
brew tap alan890104/sumi
brew install --cask sumi
```

## Update

```bash
brew upgrade --cask sumi
```

## Uninstall

```bash
brew uninstall --cask sumi
```

To also remove user data and caches:

```bash
brew zap --cask sumi
```

## Requirements

- macOS (Apple Silicon / arm64)
- [Homebrew](https://brew.sh/)
